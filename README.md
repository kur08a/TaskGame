# TaskGame (Android, Offline)

## Let you complete the task like playing a game!

厌倦了枯燥的待办清单？受够了广告满天飞、隐私不透明的任务 APP？

**TaskGame，一款为 “拖延星人” 量身打造的本地离线任务管理器，用游戏化积分系统，把 “不得不做” 变成 “主动想做”！**

✅ **极致隐私守护**：全程本地离线运行，无网络请求、无数据上传，你的任务、密码、积分数据只存在你自己的手机里，卸载即清，绝不泄露。

## 简介

本项目为纯离线单机任务管理应用，核心包含：

- 可选本地密码解锁（6-16 位）
- 任务管理（待开启 / 进行中 / 已完成 / 已逾期）
- 积分激励与逾期扣分、任务复活
- 本地 SQLite 加密存储（SQLCipher + Room）

## 运行方式

1. 用 Android Studio 打开项目根目录 `TaskGame`
2. 等待 Gradle 同步完成
3. 连接设备或启动模拟器
4. 直接运行 `app` 模块

## 简单使用方法

- 直接下载 TaskGame.apk 安装到手机上
- 可在个人中心设置用户名、开启或关闭密码保护、修改密码等
- 点击“+新建任务”创建新任务
- 创建任务时必须填写任务名称、任务难度、截止日期、任务优先级，可选填任务描述、添加多个子任务(可选的)
- 任务展示界面的每个任务将按照截止时间先后排序，截止时间相同的按任务优先级排序
- 点击任务进入任务详情页，可查看任务描述、子任务、完成进度、当前任务操作按钮等
- 任务逾期自动按任务难度扣除相应积分，可使用积分将任务复活，初始积分100

## 说明

- 无网络权限、无登录、无云同步
- 所有数据仅保存在本地
- 卸载应用后数据会被系统清除
